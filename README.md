# Contact Book

Einfaches Kontaktverwaltungssystem im Browser mit HTML, CSS und JavaScript.

## Funktionen:
- Kontaktname und Telefonnummer hinzufügen
- Kontakte anzeigen und löschen
- Responsive Benutzeroberfläche

## Technologien:
- HTML
- CSS
- JavaScript (DOM Manipulation)
